## installation

How to install this tool:

- install Release.zip
- Unzip Release
- run install.bat
- open Powershell and run elx setup

## Usage

if installing tools for git:

- open Powershell
- run elx install
- chose git

if installing libarys in C#:

- go to project folder
- open in terminal
- run elx install
- chose the libary you want to use
